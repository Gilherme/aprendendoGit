Comandos

Para iniciar um repositório e adicionar arquivos a ele

  • git init => Inicia um repositório, de um Git Bash Here e no prompt digite git init para criar um repositório
  • git status => Da uma atualização geral do seu rpositório principalmete para saber se novas versoes foram adicionadas
  • git add "nomeDoCoigo" => Prepara o arquivo para ser adivcionado ao controle de versão (ao git hub)
  • git add . => Prepara todos os arquivos para o controle de versão
  • git commit -m "Resumo da nova versão" => da um nome a nova versão que está sendo criada, isso é uma ótima boa pratica
  • git remote add origin https://github.com/Gilherme/linkDoSeuRepositório => Esse comado da a localização do repositório a ser usado, é importante que o repositório já exista
  • git push --set-upstream origin nomeDaBranch => Esse comando é nescessario para definir em qual branch a versão será "alocada", guardada... normalmente já está definida como master, e se vc der um git push antes, vai aparecer esse mesmo comando
  • git push => O git push por fim envia ao git hub a nova versão

  • git bash here => depois de feixar o comando do git, para reabri-lo basta clicar com o botão sireito do mouse dentro da pasta que está seu arquivo e dar um git bash here
  • git status => importante para ter uma noção do que não está atualizado, etc. Depois disso pode-se usar os comandos acima para adicionar novas versões
  • git reflog => para ver todas as versões anteriores, a versão mais atual sempre será a do topo
  • git reset --hard ID => Para voltar a uma versão antiga, o ID é o que aparece de amarelo no começo da linha quando da o git reflog
